EDA Process Summary

The exploratory data analysis (EDA) process followed these key steps:

  1. Data Cleaning and Preparation: Raw data from all sources was cleaned, missing values were handled, and outliers were identified and addressed.
  2. Individual Dataset Analysis: Each data source (WiFi, location, GPS) was analyzed separately to understand its characteristics, distributions, and key patterns.
  3. Feature Engineering: New features were created from the raw data to better capture the patterns and relationships in the data.
  4. Spatial Analysis: Geographical patterns and clusters were identified and mapped to understand the spatial distribution of coverage quality.
  5. Temporal Analysis: Time-based patterns and variations in signal quality were analyzed to understand how coverage changes over time.
  6. Data Integration: All data sources were merged to create a comprehensive view of the coverage patterns and prepare for modeling.
